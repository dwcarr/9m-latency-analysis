System ID Step Response Exploration

This page uses the preserved step-response subset written to system_id_step_responses.csv. The subset requires a step-like target and excludes final-position arrival measurements below 50 ms, because those near-zero points were ruled out as invalid for this data.

Velocity Capacity

Maximum velocity magnitude is the peak absolute finite-difference velocity of the actual motor position after the final target update and before the next target update, capped by the configured response window.

Velocity Rise vs Starting Angle

Velocity rise time is the first post-command time where absolute velocity reaches 90% of that movement's own peak velocity. Plotting it against starting actual angle can expose angle-dependent delay, backlash, or slop.
